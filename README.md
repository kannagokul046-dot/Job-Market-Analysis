# Job Market Analysis Dashboard

## Objective
To analyze job market trends using real job listing data and identify in-demand skills, hiring locations, job roles, and experience requirements.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI
- Excel

## Project Workflow
1. Collected raw job market data
2. Cleaned and processed the data using Python
3. Performed analysis using SQL queries
4. Built an interactive Power BI dashboard
5. Used DAX to create global KPIs and skill-level analysis
6. Added slicers for dynamic filtering and trend analysis

## Key Insights
- Identified top hiring locations across India
- Found the most in-demand technical skills
- Analyzed job demand by experience level
- Observed job posting trends over time

## Files Included
- Job_Market_Analysis_PowerBI.pbix – Power BI dashboard
- cleaned_naukri_job_market.xlsx – Cleaned dataset
